<h1>FastFeet Mobile</h1>

Este projeto faz parte do desafio final do curso GoStack da Rocketseat

**Descrição**: Este projeto é o mobile da aplicação Fasfeet, uma entregadora ficticia idealizada pela Rocketseat para o projeto de conclusão do curso, este projeto em especifico foi desenvolvido exclusivamente para o entregador para que o mesmo possa:

 Para rodar a aplicação android - `react-native run-android`
 Para rodar a aplicação ios - `react-native run-ios`
